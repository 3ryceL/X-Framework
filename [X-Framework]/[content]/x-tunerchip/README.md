# x-tunerchip
Vehicle Tuning System for X-Core Framework ::

## Dependencies
- [x-core](https://github.com/xcore-framework/x-core)
- [x-hud](https://github.com/xcore-framework/x-hud) - For NOS status display

## Features
- Vehicle tuning with Tunerchip (Item name: tunerlaptop)
- Usable NOS for vehicles (Item name: nitrous)

## Installation
### Manual
- Download the script and put it in the `[x]` directory.
- Add the following code to your server.cfg/resouces.cfg
```
ensure x-core
ensure x-hud
ensure x-tunerchip
```
